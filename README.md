# AI Procurement Intelligence System

## Overview
This project analyzes procurement data using AI-driven analytics and an interactive dashboard.

## Features
- Spending Analysis
- Supplier Insights
- Risk Detection
- Forecasting
- Chat-based AI Agent

## Tech Stack
- Python
- Pandas
- Plotly
- Streamlit

## Phases
1. Data Cleaning
2. Analytics Engine
3. AI Agent
4. Streamlit Dashboard

## Run
streamlit run app.py
